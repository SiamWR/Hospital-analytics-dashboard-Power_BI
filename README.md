# Hospital-analytics-dashboard-Power_BI
Interactive Power BI dashboard for hospital performance monitoring, patient demographics, healthcare trends, and treatment cost analysis.
## Overview

This project presents an interactive Hospital Monitoring Dashboard developed using Power BI. The dashboard provides comprehensive insights into hospital operations, patient demographics, healthcare trends, and treatment costs through dynamic visualizations and KPI monitoring.

The objective of this dashboard is to support healthcare decision-making by transforming raw hospital data into actionable insights for administrators and healthcare professionals.
## Key Features

### Patient Demographics
- Admitted patient monitoring
- Age distribution analysis
- Gender analysis
- Condition-based patient segmentation
### Key Trends
- Admission trends over time
- Length of Stay (LOS) analysis
- Day-type admission analysis
- Billing trend monitoring
### Treatment Cost Analysis
- Revenue contribution by hospitals
- Treatment cost by admission type
- Cost analysis by age group
- Hospital-wise financial performance
### Interactive Components
- Year filters
- Condition filters
- Hospital filters
- Dynamic drill-down visualizations
## Tools Used

- Power BI
- DAX
- Power Query
- Data Modeling
- Interactive Visualizations
- Healthcare Analytics
